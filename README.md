```sh
├─.obsidian
│  ├─plugins
│  └─themes
├─_figure
├─_helper
└─_template
```

```sh
.obsidian：目录为该库的配置文件，包括设置、插件、主题等；
_figure：图片;
_helper：该目录下放置了markdown的基本语法文档，下载自obsidian中文论坛;
_software：该目录下放置了最新的Obsidian软件；
_template：模板
```

## 模板使用
1. 首先安装 `_software` 目录下的软件；
2. 模板作为库文件夹，使用Obsidian打开；
